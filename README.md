## 🇬🇧 English

I'm a software engineer with **7 years of commercial experience in 1C** — enterprise
automation, ERP-style business logic, integrations, and platform-specific tooling
(see [GetManHTTP](https://github.com/LunatikDG/GetManHTTP), an HTTP client built

**What I bring:**

- A builder's mindset — I finish what I start (see project history below)
- Currently building genuine commercial-grade experience in Go, one project at a time

**Currently learning / building:**
- 🔭 Working on backend projects in Go (in progress — links coming soon)
- 📝 Writing about the transition from enterprise/1C thinking to modern backend
  development
- 🌱 Also exploring Java

**Reach out if:** you want to talk about enterprise automation, backend
architecture, or you're doing something interesting in Go and want a second pair
of eyes.
